# PES 2015 AI Documentation: teamId.txt

This is a technical configuration file that maps team-side logic (Home/Away) to the AI engine's internal IDs.

---

## 1. Constants
- **home**: The internal ID used to identify the Home team (Default: 32).
- **away**: The internal ID used to identify the Away team (Default: 16).
- **useJson**: Determines if the engine should override these IDs with external JSON data.
